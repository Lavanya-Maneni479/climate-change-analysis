# 🌍 Climate Change Analysis (1900-2021)

An end-to-end data analysis project analyzing global temperature trends, 
CO2 emissions, and natural disasters over 120 years.

## 📊 Tools & Technologies
- **Python** (Pandas, Matplotlib, Seaborn) — Data cleaning & visualization
- **MySQL** — Data querying & aggregation
- **Excel** — Summary tables & charts
- **Power BI** — Interactive dashboard

## 📁 Project Structure
```
climate-change-analysis/
├── Data/
│   ├── Raw/          → Original datasets
│   └── Cleaned/      → Cleaned & processed data
├── Notebooks/        → Jupyter notebook
├── SQL/              → SQL queries
├── outputs/          → Charts & dashboards
└── climate_analysis.xlsx → Excel summary
```
## Key Findings
- 🌡️ Global temperature rose by **+1.36°C** (1900-2021)
- 🔥 Hottest year on record: **2015 (9.83°C)**
- 🏭 Top CO2 emitter: **United States (284,036 Million KT)**
- 🌊 Most frequent disaster: **Flood (5,551 events)**
- ☠️ Deadliest disaster: **Drought (11.7M deaths)**
- 📈 Total disasters recorded: **16,126**

## 📂 Data Sources
- Berkeley Earth Temperature Dataset (Kaggle)
- World Bank CO2 Emissions Dataset (Kaggle)
- EM-DAT Natural Disasters Database (Kaggle)
